## 1. PRODUCT KNOWLEDGE

### What is Femoflor®?
Femoflor® is a real-time PCR (qPCR) diagnostic kit family for **qualitative and quantitative evaluation of the female urogenital tract microbiota**. The system uses multiplex real-time PCR to detect and quantify DNA of:
- Normal microbiota (Lactobacillus spp., Bifidobacterium spp.)
- Opportunistic pathogens (aerobes, anaerobes, mycoplasmas)
- Obligate pathogens (STIs: Chlamydia, Gonorrhea, Trichomonas, etc.)
- Herpes viruses (HSV-1, HSV-2, CMV)
- Human Papillomaviruses (HPV — oncogenic and non-oncogenic types)

### Kit Variants
| Kit Name | Description |
|----------|-------------|
| Femoflor® Screen | Basic screening panel |
| Femoflor® 16 | Extended 16-biocenosis panel |
| Femoflor® II | Second-generation comprehensive panel |
| Femoflor® AlphaScreen | Specific screening variant |
| Femoflor® DeltaScreen | Specific screening variant |
| Androflor® | Male urogenital tract microbiota |
| Androflor® Screen | Male screening panel |

### How It Works
1. **Sample collection** — vaginal (V) or cervical (C) swab
2. **DNA extraction** — nucleic acid isolation from clinical sample
3. **Real-time PCR amplification** — multiplex qPCR with fluorescent detection
4. **Software analysis** — automated quantification (lg GE/mL) and ratio calculation
5. **Report generation** — clinical interpretation with microbiota status assessment

## 4. TRANSLATION CONVENTIONS & STYLE DECISIONS

### Standardized Term Choices

The following terms were standardized after review. **Always use the preferred term:**

| Concept | PREFERRED Turkish | AVOID |
|---------|-------------------|-------|
| Relative amount/quantity | Göreceli miktar | Göreceli bolluk, bağıl bolluk, göreli bolluk |
| Proportion/share | Oran | Pay |
| Detection (combined) | Birleşik tespit / Toplu saptama | Birleşik belirleme |
| Test result interpretation | Test sonuç değerlendirmesi | Nihai değerlendirme |
| Sample type | Örnek türü | Alım lokusu |
| Unknown etiology | Etiyolojisi bilinmeyen | Etiyolojisi belirlenemeyen |
| Mixed dysbiosis | Karışık disbiyoz | Karma disbiyoz |
| Opportunistic microorganisms | Fırsatçı mikroorganizmalar | Fırsatçıl (TYPO) |
| Quantification accuracy | Kantitasyon doğruluğu | Kantifikasyon doğruluğu |
| Within reference range | Referans aralığında | Normal sınırlar içinde |

### Grammar & Style Rules for Turkish Medical Software

1. **Formal register (siz/yazılı dili):** Use formal imperative forms (-ınız/-iniz): "Bakınız", "Değerlendiriniz"
2. **Passive constructions preferred:** "Saptandı" (was detected), "Değerlendirilmesi yapılamaz" (assessment cannot be performed)
3. **Abbreviations:** Keep internationally recognized abbreviations in English (DNA, PCR, HPV, STI/CYBE, GE/mL)
4. **Species names:** Always in italics: *Lactobacillus* spp., *Gardnerella vaginalis*
5. **HTML tags:** Preserve all HTML formatting tags exactly (<b>, <i>, <sup>, <br>, &nbsp;, &lt;, &gt;)
6. **Decimal separator:** Turkish uses comma (virgül): 4,0 NOT 4.0 — but the software B_DOT string handles this
7. **Percentage symbol:** Turkish convention: %80 (percent before number) BUT in scientific context both %80 and 80% are acceptable
8. **No translation of:** Product names (FEMOFLOR®, Androflor®), unit abbreviations (lg, GE/mL), organism names

### Common Warning/Note Patterns

```
<b>NB!</b> [Warning in Turkish].
<b>ÖNEMLİ!</b> [Important notice in Turkish].
Dikkat! [Attention notice in Turkish]
```

## 5. KNOWN ISSUES & PITFALLS

### Cyrillic Characters in Source
The English source text occasionally contains **Cyrillic characters** that look identical to Latin characters (e.g., Cyrillic "С" instead of Latin "C" in "Сontrol"). These must be replaced with proper Latin characters in all translations.

### Inconsistencies in Original Translation
The first-version Turkish translation contains several inconsistencies that were identified and corrected:
- Mixed use of "bolluk" / "miktar" for "amount"
- Mixed use of "pay" / "oran" for "proportion"
- Occasional typos ("fırsatçıl" instead of "fırsatçı", "Göeceli" instead of "Göreceli")
- "Kantifikasyon" (incorrect borrowing) instead of "Kantitasyon"

## 6. REGULATORY CONTEXT

### TİTCK Requirements for Software Localization
- All user-facing software text must be in Turkish per Turkish IVD regulation (harmonized with EU IVDR 2017/746)
- Technical/scientific abbreviations (DNA, PCR, HPV) may remain in English/Latin
- Product names (FEMOFLOR®) are not translated
- Clinical interpretations must be clear and unambiguous for healthcare professionals
- ÜTS (Product Tracking System) registration requires Turkish-language documentation

### IVDR Classification
Femoflor® / Androflor® kits are classified as **IVD medical devices** requiring:
- CE marking (CE işareti)
- Performance evaluation documentation
- Quality management system (ISO 13485)
- Post-market surveillance
